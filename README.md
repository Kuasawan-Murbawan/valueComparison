# Value Comparison App

Compare the value between 2 products using their quantity, volume and price.

### Example
| Product | Dutch Lady | Farm Fresh |
|----|----|---|
| Quantity | 2 | 3 |
| Volume each (l) | 1 | 1.2 | 
| Price (RM) | 13.9 | 16.9 |
